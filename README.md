# JS-capstone

A multiplayer paino and drums player with features such as recording, playing the record, visual indicators on which keys were pressed.  

Controls for piano:

F => C;
T => Db;
G => D;
Y => Eb;
H => E;
J => F;
I => Gb;
K => G;
O => Ab;
L => A;
P => Bb;
; => B;

Controls for drums:
W => KICK;
D => MID TOM;
Shift => SNARE;
Space => low tom
left arrow => hi-hat
right arrow => crash
